3. Skills, qualifications, and experience requirements:

3.1 Hard skills:

HTML stands for HyperText Markup Language. In this language, each element will be stored within open and close tags and have its properties. We can easily control each element like a node in Object-Oriented Programming. Every web developer must know this language.
CSS stands for Cascading Style Sheets. Generally, there a two primary missions of CSS files. Firstly, It identifies the style of each element, such as color, background color, font family, opacity, etc. Another one is setting up the priority of each element and modify the original preference. There are several methods to control the display of the layout such as Flexbox, grid, or classes of bootstrap 4. CSS is a complicated programming language. However, it is necessary to build the first foundation.
The majority of web developers are familiar with JavaScript language because of this utility. JavaScript can get particular elements quickly, and then It can handle and modify the attributes of this element. Front-End Developers can handle the event by JavaScript. So, JavaScript can execute the events and interact well with tags in the HTML files.
Responsive Web Design: Nowadays, more and more electronic devices are invented. This phenomenon comes with an increment of number screen sizes using to access websites. Front-end developers play a vitally important role in design websites whose display is relevant to different sizes of various devices.
JS Front-end Frameworks: To boost the performance of websites, scale up the projects as well as make readable code.JS Front-end Frameworks such as Vue, React, or Angular should be considered.

My Project Ideal

Overview:

The environment campaign is always considered by every people. My project is making a website contains the environment information, articles, e-news. Furthermore, To give the green organization a solution that can resolve the battery pollution problems. My web application uses free APIs service about environment news to upload itself constantly. The writers also upload their knowledge and campaign to share the information about the current conditions of our environment or call other users to give a helping hand for their following environmental campaign

 

Motivation:

1.Batteries are small, easy to collect but It caused a huge damage for our environment as well as the health of human.
According to http://www.batterycompany.co.uk/ , There are 660 million batteries bought each year in the UK, that’s an average of 21 per household.
Likewise, The money spend on batteries up to 5 million dollars each week but 98% of them was not recycled.
2.Batteries contain the heavy metals that can easily pollute water. If people drink this water, those heavy metals directly influence the human health.
3.Batteries can cause huge damage but easy to collect. This reason motivated me to make an web app that can give a solution for this problem.

Description:

My web app have several features for people who concern environmental problems
1. We have a homepage that people can read the newspaper and the information about the current conditions of our environment. They can comment, like, share their knowledge ,and give their opinion. This homepage take information from some APIs server and reload itself every 15 minutes.
2. The Page that connects with angel investors, advertise and sell the products of those brands to make a fund for operation.
3. The Login feature: With users: My flatform can give a form to customers. Users have a responsibility to fill the form about their location, their number of batteries. My web app will return the nearest collecting places.
With connected organizations: They can see how many batteries in each store and set up a plan to gather.
4. Collecting places: Firstly, We guidance citizens on how to save the batteries. (like this images :D :D :D) I will choose and connect with specific convenient stores. People can give their batteries which were out of energy to those places. We also connect with a particular green organization to have needed facilities and policies.
